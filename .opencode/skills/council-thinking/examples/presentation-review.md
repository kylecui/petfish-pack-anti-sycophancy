# Worked Example: Presentation Mainline Review

## User Input

```text
用 Council 审查这个 PPT 主线。

我打算做一个投资人路演 PPT，结构如下：
1. 市场规模巨大
2. 我们团队很强
3. 产品技术领先
4. 商业模式清晰
5. 融资用于扩张
```

## Council 判断

### 1. 问题重述

真正要判断的是：这份路演 PPT 的主线是否能有效降低投资人的疑虑，并推动他们进入下一步尽调。

---

### 2. 五个顾问的判断

#### 反对者

- **核心判断**：这个结构是典型的"自我表扬式"主线，没有解释为什么"是现在"、"是你"、"为什么投资人应该信"。
- **关键理由**：
  - "市场巨大"只会引发"那巨头为什么不来做"。
  - "团队很强"没有证据，投资人见过太多强团队。
  - "技术领先"没有定义领先标准和验证方式。
- **对决策的影响**：PPT 会让投资人把重点放在挑错上，而不是被说服。
- **我不知道**：是否有已有客户收入或增长数据。

#### 本质思考者

- **核心判断**：投资人真正要问的不是"这个项目好不好"，而是"我如果错过这个项目，损失有多大"。
- **关键理由**：
  - 市场大不等于机会属于你。
  - 投资决策的关键是稀缺性和时机，而不是全面性。
  - 主线应围绕"为什么这个机会窗口只属于我们"展开。
- **对决策的影响**：PPT 顺序需要调整，开场应讲清稀缺性和时机，而不是市场规模。
- **我不知道**：是否有独占资源、专利或排他合作关系。

#### 机会挖掘者

- **核心判断**：可以借势"巨头不做细分市场"这一点，把"强团队"包装成"唯一懂中小企业场景的团队"。
- **关键理由**：
  - 如果市场确实巨大但巨头忽视，这是有利叙事。
  - 团队背景如果真能证明对场景的深刻理解，就是护城河。
  - 商业模式清晰可以压缩成一页"单位经济模型"。
- **对决策的影响**：主线可以做成"被忽视但正在变大的市场 + 唯一能做成的人 + 已被验证的小闭环"。
- **我不知道**：团队是否有其他竞争对手无法复制的具体经历。

#### 局外人

- **核心判断**：外部听众看完前 3 页后只会记住"又是一个自我感觉良好的创业团队"。
- **关键理由**：
  - 投资人每天看大量相似的 PPT，"团队很强""市场很大"是噪音。
  - 第 1 页就谈市场规模，会让人默认你是因为市场大才进来，不是因为真实洞察。
  - 没有客户 logo、没有数据、没有真实痛点引用。
- **对决策的影响**：需要把"证据"前置，把"自我介绍"后置或压缩。
- **我不知道**：是否有可展示的客户名称或数字。

#### 执行者

- **核心判断**：在打磨 PPT 之前，先补一个投资人听过之后的反馈清单。
- **关键理由**：
  - 当前结构的问题是"自嗨"，最该验证的是投资人看到第 3 页后提出的前三个问题。
  - 不是每页都要改，而是要先确认哪一页留下记忆点。
- **对决策的影响**：先做 3 次 mock pitch，收集投资人疑问，再改 PPT。
- **我不知道**：路演的总时长和投资人背景。

---

### 3. 交叉审查

**重复或低价值观点**：
- 反对者和局外人都指出 PPT 是自我表扬式结构——合并。
- 本质思考者和机会挖掘者都提到"独有""稀缺"——合并为"时机与稀缺性"主题。

**真正有价值的观点**：
- PPT 主线应回答"为什么是现在、为什么是你"，而不是全面介绍项目。
- "团队很强"需要转化为"为什么这个团队能做成别人做不成的事"。
- 投资人疑虑应在 PPT 中主动回应，而不是等他挑错。
- 应该有真实证据（客户、数据、单位经济）。

---

### 4. 去掉弱观点

**删除**：
- "市场规模巨大"作为开场页。
- "产品技术领先"空泛口号。
- "融资用于扩张"放在主线前面。

**保留**：
- 用"为什么这个机会只属于我们"作为主线锚点。
- 用"团队是这个细分市场唯一能做成的人"替代"我们团队很强"。
- 用"小闭环已被验证"替代"商业模式清晰"。

---

### 5. 综合结论

**我的判断是**：当前 PPT 主线是典型的自嗨式路演结构，投资人大概率无感。建议改成以"稀缺性 + 时机 + 小闭环验证"为主线的故事。

**最大风险是**：投资人听完前 5 分钟后没有任何记忆点，路演后不会被推进尽调。

**最大机会是**：如果能把"巨头忽视的细分场景 + 团队独特理解 + 已被验证的小闭环"讲清楚，可以在同品类中脱颖而出。

**下一步应该**：
1. _BEFORE_ 做 PPT 美化，先做 3 次目标投资人 mock pitch，记录前三个问题。
2. 把"市场规模"页改成"为什么是现在"（包含行业变化、政策变化、技术变化中的至少一个）。
3. 新增一页"我们验证了什么"，列出具体客户、收入、使用效果数据。

**我不知道的是**：
- 是否有已付费或签约的早期客户。
- 投资人通常的拒绝理由是什么。
